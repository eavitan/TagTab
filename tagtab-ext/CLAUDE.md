## Extension Directory
This directory contains the actual browser extension files. When working on the extension code, refer to the parent directory's documentation and task planning.

## Key Files in This Directory
- `manifest.json` - Extension configuration
- `background.js` - Service worker with core functionality
- `popup.html/js/css` - Toolbar popup interface
- `pages.html/js/css` - Tab management interface
- `icons/` - Extension icons
